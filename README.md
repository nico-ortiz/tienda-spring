# Spring Boot CRUD Application 

##  Requirements

For building and running the application you need:

- Java Version 17.0+
- Maven 3

## How to run 

Run command: 

```
./mvnw spring-boot:run
```
Run tests:

```
./mvnw test
```

To run especific test:
```
./mvnw test -Dtest="<test class name>"
```
